Finreport

Автоматизирует массовое скачивание финансовых отчётов из личного кабинета поставщика Wildberries: собирает все доступные отчёты за выбранный период в единый архив и выгружает прочие удержания в Excel. Поддерживаются отчёты, доступные в кабинете, начиная с 29.01.2024.

Массовое скачивание
Собирает отчёты за указанный период и упаковывает их в ZIP-архив.
Удержания в Excel
Формирует выгрузку прочих удержаний в формате .xlsx для последующего анализа.
Просмотр и выгрузка
Расширение позволяет в отдельной вкладке браузера просматривать и выгружать необходимые отчеты ...
Статус: активная разработка
Платформа: Chrome/Chromium-браузеры
Скачать расширение ⬇ Открыть на GitHub
Как установить в Chrome
  1. Скачайте ZIP-архив и распакуйте его.
  2. Откройте: chrome://extensions/
  3. Включите «Режим разработчика» (Developer mode) справа вверху.
  4. Нажмите «Загрузить распакованное» (Load unpacked).
  5. Выберите папку с файлами расширения (где находится manifest.json).

WBin

Инструмент для парсинга рекламных кампаний Wildberries и связанных с ними артикулов, а также для сбора и агрегирования статистики по кампаниям прямо из личного кабинета.

Парсинг кампаний
Извлекает структуру рекламных кампаний, группы, ключевые фразы и привязанные артикулы.
Статистика из ЛК
Собирает метрики эффективности кампаний из личного кабинета маркетплейса.
Экспорт данных
Подготовка данных к анализу: CSV/JSON экспорт для BI и табличных инструментов.

Google Sheets / Apps Script

Sheetswbparser — 🔍 Парсер позиций Wildberries

Отслеживайте позиции ваших товаров в поиске Wildberries прямо в Google Таблицах с помощью пользовательской функции.

Формула
=FIND_WB_POSITION(A2; B2)
Простая интеграция
Код вставляется в Apps Script файла таблицы.
Фокус на задаче
Минимум настроек — используйте как обычную функцию.
Статус: рабочая версия
Платформа: Google Sheets
Открыть на GitHub Открыть скрипт
Как пользоваться
  1. Откройте в таблице: Расширения → Apps Script.
  2. Скопируйте код проекта в редактор и сохраните. Разрешите доступ при запросе.
  3. В ячейке используйте формулу: =FIND_WB_POSITION(A2; B2).
  4. Где: A2 — артикул, B2 — поисковый запрос. Результат (позиция) выведите в столбец C.
