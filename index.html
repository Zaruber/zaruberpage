Sheetswbparser — 🔍 Парсер позиций Wildberries

Парсер позиций для Google Таблиц с новыми возможностями: парсинг по рекламным полкам конкурентов и выбор конкретного города.

Формула
=FIND_WB_POSITION(A2; B2)
Новые сценарии парсинга
Добавлен парсинг по рекламным полкам конкурентов для более точного сравнения.
Гео-точность
Есть выбор конкретного города для проверки позиций в нужном регионе.

Как пользоваться
  1. Откройте в таблице: Расширения → Apps Script.
  2. Скопируйте код проекта в редактор и сохраните. Разрешите доступ при запросе.
  3. В ячейке используйте формулу: =FIND_WB_POSITION(A2; B2).
  4. Где: A2 — артикул, B2 — поисковый запрос. Результат (позиция) выведите в столбец C.
